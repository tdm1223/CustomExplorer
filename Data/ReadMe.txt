========================================================================
    정적 라이브러리 : Data 프로젝트 개요
========================================================================

응용 프로그램 마법사에서 이 Data 라이브러리 프로젝트를 만들었습니다.

프로젝트에 대해 소스 파일은 만들어지지 않았습니다.


Data.vcxproj
    응용 프로그램 마법사를 사용하여 생성한 VC++ 프로젝트의 기본 프로젝트 파일입니다. 파일을 생성한 Visual C++ 버전에 대한 정보와 응용 프로그램 마법사를 사용하여 선택한 플랫폼, 구성 및 프로젝트 기능에 대한 정보가 포함되어 있습니다.

Data.vcxproj.filters
    응용 프로그램 마법사를 사용하여 생성된 VC++ 프로젝트의 필터 파일입니다. 이 파일에는 프로젝트의 파일과 필터 간의 연결 정보가 들어 있습니다. 이러한 연결은 특정 노드에서 유사한 확장명으로 그룹화된 파일을 표시하기 위해 IDE에서 사용됩니다. 예를 들어 ".cpp" 파일은 "소스 파일" 필터와 연결되어 있습니다.

/////////////////////////////////////////////////////////////////////////////
컴파일러와 링커 스위치가 MFC를 지원할 수 있도록 변경되었습니다. 이 프로젝트에서 MFC 클래스 마법사를 사용하려면 프로젝트에 "resource.h", "Data.rc" 및 resource.h.가 포함된 "Data.h" 등의 여러 파일을 추가해야 합니다. 정적 라이브러리에 rc 파일을 추가하면 DLL 또는 EXE에는 rc 파일이 하나만 있어야 하는 제약 조건 때문에 어려움을 겪을 수 있습니다. 이 문제를 해결하려면 해당 라이브러리의 .rc 파일을부모 프로젝트의 .rc 파일에 추가하면 됩니다.

응용 프로그램 마법사에서 사용하는 "TODO:" 주석은 사용자가 추가하거나 사용자 지정해야 하는 소스 코드 부분을 나타냅니다.
